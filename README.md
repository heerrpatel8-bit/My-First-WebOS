# 🖥️ My First WebOS

A browser-based desktop environment inspired by modern operating systems. **My First WebOS** recreates the look and feel of a desktop OS directly in the browser, featuring an interactive interface, draggable windows, desktop icons, and a responsive user experience built using web technologies.

## 🌐 Live Demo

👉 https://heerrpatel8-bit.github.io/My-First-WebOS/

---

## ✨ Features

* 🖥️ Desktop-style user interface
* 📂 Interactive desktop icons
* 🪟 Window-based application layout
* 🎨 Modern and responsive design
* ⚡ Smooth user interactions
* 📱 Works across modern web browsers

---
